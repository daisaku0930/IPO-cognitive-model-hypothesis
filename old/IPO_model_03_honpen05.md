# 05 ― 起点と創造：どこから読み始めるか

## 当然の前提を、ひっくり返す

ここまで、流れを必ず **I → P → O** の向きで読んできた。
だが、よく考えるとその向きは「正しい順番」だから選ばれたのではない。**観測者が、どこを起点に置いたか**で決まっていただけだ。

## 流れは、輪の一部

一つのIPOの結果（O）は、たいてい次の何かの材料（I）になる。
だから本当は、流れに「始まり」も「終わり」もない。

```
… → O → I → P → O → I → P → O → …
```

この連続のどこかに線を引いて「ここが Input」と呼ぶ――その線の引き方が観測者の選択だ。【思弁】

## OPI ― 結果から読む

新人と先輩で考える。

- 普通の向き（IPO）：材料を見て（I）→ 考えて（P）→ 答えを出す（O）
- 逆の向き（OPI）：まず「こういう結果がほしい」（O）から始め → そこへ至る過程を逆算し（P）→ 必要な材料を取りに行く（I）

OPIは別の能力ではない。**同じ流れを、違う起点から読んでいるだけ**だ。
（だからO型は「受け身で答えだけ欲しい人」とは限らない。欲しいOから逆算する人もまたO起点だ。）

## 創造 ＝ 起点を選ぶ自由度

起点が一つに固定され、いつも材料（I）からしか入れないと、**与えられた材料の組み合わせ**しか出てこない。

起点を自由に選べると――欲しい結果（O）から、あるいは途中の処理（P）から――流れを組み直せる。
これが「創造的」と呼ばれるものの一面だと読める。【思弁】

本編04の新人B（「この手順の、ここが穴だ」）を思い出すと、彼は結果から構造へ**起点を移せた**人だった、とも言える。

## 正直な位置づけ ― ここは理論ではなく思想

前の節（W・n・m・補正・更新）には、曲がりなりにも検証可能な予測の芽があった。
だが**この節は違う**。

起点の自由度は、外から観察しても判別できない。
「超高速で IPO を回している人」と「OPI で逆算している人」は、振る舞いの上で区別がつかない――しばしば本人にも区別できない。
つまり**反証可能性はほぼゼロ**だ。【思弁】

だからこの節は、「測れる主張」としてではなく、**ものの見方を一つ増やす装置**として置く。
価値は予測力（当てる力）ではなく、生成力（アイデアを生む力）の側にある。理論の核ではなく、思考の自由度を上げる思想として読んでほしい。

## 本編を閉じる

通してきた骨格はこうだ。

```
W       どこを見るか        （01）
n・m    どれだけ細かく・深く （02）
α・Θ・Δ どう行間を埋めるか   （03）
更新     どう変わるか        （04）
起点     どこから読むか      （05）
```

最後の一手（起点）が効かせるのは、この座標系を「**固定された地図**」ではなく「**回し方・読み方を選べる装置**」に変えることだ。

C は認知状態の*記述*であると同時に、*操作できる対象*でもある――この二面性が、本編の結論にあたる。

> ここから先（③使い方）は、この「操作できる」という性質を、設計・AI・運用の具体的な道具に変えていくパートになる。

## 座標系への接続

```
IPO × C

C = (W, n, m | α, Θ, Δ)
```

この節が動かすのは、署名の **IPO（流れ）の因子**――状態 C（2行目）の中身は増えず、増えるのは「IPO をどの向き・どの起点で回すか」という操作の自由度のほうだ。定義は**正典**を参照。
