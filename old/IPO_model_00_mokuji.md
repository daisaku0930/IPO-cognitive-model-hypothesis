# IPO認知モデル ― 目次と読み方

> **一言で**：人の認知を Input → Process → Output の流れとして捉え、いまの状態を座標 C で表す。
> モデルの全体像は「**流れ × 状態**」＝ `IPO × C`（状態 C の中身は `C = (W, n, m | α, Θ, Δ)`）。IPO と C は階層が違うので署名と定義を分けて書く。
> 「理論」ではなく「**枠組み ＋ 使える道具**」として読む。

## 読み方のルール（全ファイル共通）

- **定義は正典だけが持つ**。本編・使い方は参照するのみで、再定義しない。
- **構造と理論の線引きはレイヤー注記が持つ**。本文では語らない（語ると膨らむ）。
- **新人（O寄り）と先輩（P寄り）** が全編を通したテストケース。変数で迷ったらこの二人に戻す。
- **「型」は場面で振れる状態の略記**（正典1節）。人の固定ラベルではない。
- **確信度タグ**：【確立】＞【仮説】＞【思弁】。【思弁】の主張を断定に使わない。
- **評価軸が部で変わる**：本編は「真か（妥当性）」、使い方は「使えるか（有効性）」、付録Aは「外れる形になっているか」で読む。

## このモデルの現在地（正直に）

- 「理論」ではなく **観察フレームワーク ＋ 操作ツール**。構造（軸）の上に、外れうる理論（リンク・法則）が薄く乗る。
- **形式化**（記号・式がある）はされているが、**操作化**（測定手段がある）はされていない。
- 第一の効用は予測力でも生成力でもなく **弁別力**――見えていなかった軸を指せるようにすること。
- 地に足を着ける唯一の経路は**外に出ること**（n・m が遠い相手で試す）。作者一人では完成しない設計。
