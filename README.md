# IPO認知モデル ― 原典

> 人間の認知構造を工学的に捉え直すための、出発点の地図。

---

## これは何か

人は同じ状況を見ていても、**どこに注意を向けるか**が異なる。

このモデルは、人間のあらゆる認知をInput→Process→Outputの流れとして捉え、その違いを観察・記述・操作するためのフレームワークです。「理論」ではなく観察フレームワーク。形式化（記号がある）はされていますが、操作化（測定手段がある）はされていません。

---

## モデルの構造

```
IPO × C

C = (W, n, m | α, Θ, Δ)
```

- **IPO**：流れの軸（Input → Process → Output）
- **C**：状態の軸（いまの認知配置）
- **縦棒の左（W, n, m）**：状態変数
- **縦棒の右（α, Θ, Δ）**：補正パラメータ（P層）

| 変数 | 意味 |
| --- | --- |
| W | 注意配分の方向（I・P・Oのどこに注意を向けるか） |
| n | 解像度ベクトル `(n_i, n_p, n_o)`（受け取り・並列・出力の細かさ） |
| m | 処理の深さ（P内部の再帰段数） |
| α | 補正の発火確率 |
| Θ | 無意識補正の閾値（①↔②の境） |
| Δ | 補正能力の上限（②↔③の境） |

---

## このモデルで説明できること

- **会話の噛み合わなさ**：能力差ではなく、WとnのズレとしてIPO構造で記述できる
- **なぜ教え方が刺さらないのか**：送り手と受け手のn・mの差がΔで補正される構造として記述できる
- **なぜ人は変わるのか**：O_intがCを自己更新し続ける動的ループ（C_t → C_t+1）として説明できる
- **創造・学習の発生条件**：中間成果物PkがI'に転じる相転移として記述できる
- **クオリア**：観測者BがAの境界面から構成する帰属（BのO）として記述できる
