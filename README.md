# IPO認知モデル仮説

> 人間の認知構造を工学的に捉え直すための、出発点の地図。

---

## これは何か

人は同じ状況を見ていても、**どこに注意を向けるか**が異なる。

このモデルは、人間のあらゆる認知をInput→Process→Outputの流れとして捉え、その違いを観察・記述・操作するためのフレームワークです。

---

## このモデルで説明できること

- **会話の噛み合わなさ**：能力差ではなく、IPOのどこを見ているかの差として構造的に説明できる
- **なぜ教え方が刺さらないのか**：教える側と学ぶ側の認知特性C=(W,n,m)のズレとして記述できる
- **なぜ人は変わるのか**：経験値（O_int）がCを自己更新し続ける動的ループとして説明できる
- **創造・学習の発生条件**：ProcessのnがmによってIPO連鎖を起こす相転移として記述できる
- **AIとの協働設計**：W・n・mをプロンプトで動的に変調する操作モデルとして機能する

---

## モデルの構造

```
I（Input）→ P（Process）→ O（Output）
```

認知特性は3軸で表現される：

```
C = (W, n, m)

W：注意配分の重みベクトル（I・P・Oのどこに注意を向けるか）
n：Processの分解能（どれだけ細かく考えるか）
m：IPO連鎖数（どれだけ深く再帰するか）
```
